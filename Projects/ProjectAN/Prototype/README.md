# ProjectAN

인기 모바일 디펜스 서브컬쳐 게임 중 하나인 '명일방주' IP를 기반으로 하는 턴제 수집형 RPG 게임으로 기획 중입니다.

'명일방주'의 리소스를 활용하여 개발하며 해당 게임은 비상업적 용도로서 온전히 Portfolio로 기능합니다.

유니티 엔진으로 개발 중이고, 상용화 게임 내 테크닉 위주로 선보일 예정입니다.

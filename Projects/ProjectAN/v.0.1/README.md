0.1 버전에서 구현한 전투 진행 방식은 자동 전투입니다.
캐릭터는 시간의 흐름에 따라 턴 포인트가 쌓이고 행동을 진행하게 됩니다. 

[주요 기술]
- TP 방식 턴제 게임의 기본 전투 메커니즘
- Spine을 활용한 애니메이션 제어
- 메테리얼을 활용한 Tint 및 Alpha의 활용
- 라운드 진출 시 스크린 연출
- UGUI와 월드 오브젝트간 연결
